# Programming Learning Center - Landing Page

A clean, responsive, and modern responsive educational landing page built from scratch using pure **HTML5** and **CSS3**. This project was created as part of a front-end web development assignment to practice structuring layouts, creating tables, managing typography, and building interactive forms.

## 🚀 Live Demo
You can view the live project here: [`click here`](https://zyad-zoz-133.github.io/courses-landing-page/)

## 🌟 Features
- **Hero Section:** Engaging header section with a background image overlay and call-to-action button.
- **About Us Section:** Clear introduction to the learning center with a structured list of benefits.
- **Our Courses:** Grid layout showcasing various courses (HTML, CSS, JavaScript, Bootstrap, PHP, Laravel) with customized action buttons.
- **Weekly Schedule Table:** A cleanly styled timetable utilizing advanced CSS selectors (`:nth-child`) for alternating row colors and distinct header accents.
- **Interactive Registration Form:** A fully structured user registration form featuring various input types (`text`, `email`, `password`, `tel`, `number`, `date`), standard radio buttons for gender selection, dynamic checkboxes for skill assessment, a custom `<select>` dropdown, and a flexible `<textarea>` box.

## 🛠️ Technologies Used
- **HTML5:** Semantic markup tags (`table`, `form`, `select`, `input`, etc.).
- **CSS3:** Custom styles, layout spacing, table styling, structural pseudo-classes, and input form responsiveness.
